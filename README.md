# JCY-Vault-Heat-Map

一个 [Obsidian](https://obsidian.md) 插件，以 GitHub 贡献图风格展示整个 vault 的文件改动记录，数据来源于所有 markdown 文件的实际修改时间。

---

## 功能

### 热力图

- 以周为列、7 天为行的网格展示 vault 全局的文件改动频率
- 顶部显示月份标签，左侧显示星期标签（一、三、五）
- 色阶共 6 级（0 = 无改动，5 = 改动最多），亮色/暗色主题分别适配
- 右下角图例直观标注色阶含义
- 工具栏实时显示统计数据：**总改动次数 · 活跃天数 · 文件总数**

### 时间范围切换

顶部提供 5 档范围按钮，切换后热力图和 Sparkline 同步刷新：

| 按钮 | 展示范围 |
|------|----------|
| 近 3 个月 | 过去 90 天 |
| 近 6 个月 | 过去 180 天 |
| 近 1 年 | 过去 365 天（默认） |
| 近 2 年 | 过去 730 天 |
| 全部 | 从最早文件日期至今 |

### Sparkline 横条

热力图下方展示与当前时间范围匹配的每日柱状图：

- 基于 p25 / p50 / p75 分位数动态计算颜色等级，与绝对数值无关
- 鼠标悬停显示日期和改动文件数
- 点击柱子 → 高亮热力图对应格子并展开文件列表

### 文件详情面板

点击热力图格子或 Sparkline 柱子后，下方弹出当天改动的文件列表：

- 按修改时间降序排列，显示具体时分
- 点击任意文件直接在新标签页打开
- 已打开过的文件显示「已访问」徽标
- 徽标状态实时刷新（打开文件时立即更新，无需手动刷新）

### 自动刷新

文件发生创建、修改、删除时，若热力图视图已打开，1 秒防抖后自动重新渲染。

---

## 打开方式

- **Ribbon 图标**：点击左侧边栏的 `activity`（波形）图标
- **命令面板**：搜索 `打开仓库改动热力图`

---

## 安装

1. 下载 `main.js`、`manifest.json`、`styles.css`
2. 在 vault 的 `.obsidian/plugins/` 目录下新建文件夹 `JCY-Vault-Heat-Map`
3. 将三个文件放入该文件夹
4. 在 Obsidian 设置 → 第三方插件 中启用 **JCY-Vault-Heat-Map**

## 色阶说明

### 热力图（6 级）

| 等级 | 含义 | 亮色主题 | 暗色主题 |
|------|------|----------|----------|
| 0 | 无改动 | 浅灰 | 深灰 |
| 1 | ≤ 15% 峰值 | `#9be9a8` | `#0e4429` |
| 2 | ≤ 35% 峰值 | `#40c463` | `#006d32` |
| 3 | ≤ 60% 峰值 | `#30a14e` | `#26a641` |
| 4 | ≤ 85% 峰值 | `#216e39` | `#39d353` |
| 5 | > 85% 峰值 | `#0d4a20` | `#56e86a` |

### Sparkline（4 级，基于分位数）

| 等级 | 条件 |
|------|------|
| 1 | count ≤ p25 |
| 2 | count ≤ p50 |
| 3 | count ≤ p75 |
| 4 | count > p75 |
